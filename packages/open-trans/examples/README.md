使用 @salad/trans 的介绍

## Installation

```
pnpm
```

Add `env.json` with config of each translator:

```json
{
  "[id]": {
    "id": "...",
    "key": "...",
    "...": "..."
  },
  "...": "..."
}
```

Build project with `pnpm build` and load extension at `OpenTranslate/examples/extension/build`.

Click the browser action icon.
